Tickey Documentation

Welcome to the official Tickey docs! Here you'll find everything you need to set up and use your ticket system.

Getting Started

To get started with Tickey, invite the bot to your server and ensure it has proper permissions.

https://discord.gg/YOUR_INVITE_HERE

Bot Setup

Use the following command to initialize Tickey in your server:

/setup

Commands

Tickey supports the following slash commands:

  • /ticket open – Opens a new support ticket
  • /ticket close – Closes the current ticket
  • /ticket add – Adds a user to the ticket

Permissions

Make sure the bot has Manage Channels, Send Messages, and Manage Roles permissions.

FAQ

Q: Does Tickey support logging?
A: Yes! Use /logs to configure your log channel.
